# Corresponding encoder source

The `foo2xqx` directory contains the exact GPL-2.0-or-later source used to
build the bundled `m1005-xqx-encode` executable. From this Source directory,
run `make phase2` with current Apple Clang to rebuild it. The top-level project
also supplies the Printer Application bridge and packaging source.

PAPPL is used under the Apache License 2.0. Its license and notice are in the
adjacent `Licenses` directory. The Phase 5 build uses PAPPL 1.4.11.
